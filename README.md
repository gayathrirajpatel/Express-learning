# Express-learning

## Express-Generator

Check versions
```bash
    node -v
    npm -v

```


Installations
```bash
    npm install -g express-generator - for earlir versions
    express --git --hbs testApp   - create testapp project with gitignore and hbs

```


Debug
```bash
set DEBUG=testApp:* & npm start
```
